ПРОЕКТ Pygame

АВТОРЫ: Степаныв Артём, Ельцов Павел

ОПИСАНИЕ:
Проект представляет собой платформер с различными уровнями, монстрами и препятствиями.
Вы управляете главным персонажем. Главная задача-дойти до определённого места, помеченного флажком.
Для того чтобы перейти на следующий уровень нужно пройти предыдущий. Также можно улучшать свои результаты, собирая звёздочки, которые находятся в сложных местах.

Стеапнов Артём - сама игра, логика игры, физика игры, монстры 
Ельцов Павел - главное меню, меню уровней, разработка уровней
